Recipes UI

The Dinner Planner is a web app we had to create in the context of Interaction Programming and the Dynamic Web. It allows the user to retrieve recipes and the corresponding groceries for the menu he/she composed. The solution is implemented following the latest industry practices. (Images are currently missing as the API from the provider has changed and access to recipes has to be adapted,)

Project Details

The course allowed us to practice programming web interfaces with AngularJs, ReactJs and VueJs, the final application has been designed and validated with usability in mind. The ANT design framework for web interfaces was used in combination with VueJs. The resulting interface was also responsive and could be used without problem cross-platform. We included error handling and user guidance.
