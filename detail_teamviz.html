TeamViz

As a first assignment for the course of Information visualization, we were given anonymised data about the students and their performance in different fields. Our task was to develop a static visualization to suppport the teacher assistant to form the best teams possible. This simple project can still be visited on https://busyxin.github.io/teamviewer/

Project Details

Designing a visualization system to support the formation of the “best groups”, can be interpreted as a tool to either build the dream team, or to form groups with the minimum disparity possible, leading to balanced groups through the whole class. Taking into consideration both use cases, I decided to utilize the radar chart to visualize individual skills as well as averages in a group.

The following skills has been grouped together as:

Math: (0.5*Mathematics + 0.5*Statistics)

Tech: (0.1*Computer Usage + 0.3*Programming + 0.2 Graphics Programming + 0,2*HCI Programming + 0,2*Code Repository)

Team: (0.5*Communication + 0.5*Collaboration)

The user is allowed to perform various operations to explore the data, mainly the combination of the table listing the students that is brushed and linked [1] to the radar chart, selecting and sorting in order to find the best fit for the group. The average of the 71 students is displayed as a reference, so the user can visually identify the strengths and weaknesses.
