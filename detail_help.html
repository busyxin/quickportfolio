121Help

121Help is a side project developed with friends to test out mobile app development. It aims to help neighboors interact and create stronger communities during this digital era that seems to sometimes set us apart.

Project Details

I worked together with a former coworker who is a designer to produce all the high fidelity layouts. We used Sketch and invisionApp, for the website, we used Middleman and Ruby on Rails. The mobile app was developed using React Native to make it compatible on iPhone and Android.
